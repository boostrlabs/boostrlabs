# 05 — Modules and Features

## Confirmed core modules

### BOOSTR Audit
Interactive public prospect qualification and diagnostic form. Captures BOOSTR Leads, not Partner Leads.

### Smart Links / Deep Links
Custom manual links for partners. Not generic Linktree clones. They should track context and intent.

### Partner Inventory
The partner’s products, services, beats, tickets, cars, cuts, apparel, content, drops, packages, and offers.

### Smart Checkout / Orders
Checkout, guest/account logic, payments, direct payment logging, order notifications, order states, receipts, post-purchase pages.

### BOOSTR Intelligence Layer
Data, signals, recommendations, heat maps, demand radar, client memory, follow-up intelligence.

### BOOSTR Follow-Up Engine
Contextual WhatsApp/email/voice note/re-engage flows.

### BOOSTR Network
Curated partner network, not an open marketplace.

### BOOSTR for Artists
Artist-focused OS and advisory module.

### BOOSTR Barber / Beauty OS
Visual service selection, packs, booking, deposits, before/after, reminders.

### Event / Drop Engine
Tickets, Smart Crowd Pricing, Claim Your Spot, live/drop room, proof of demand, sponsor page.

### Beat Store / License Engine
Beat previews, license tiers, contracts, digital signature, payments, delivery, records.
