# 10 — BOOSTR Barber / Beauty OS

## Purpose

Build a visual, mobile-first service selling flow for barbers, nail artists, and beauty partners.

## Core idea

Do not clone Booksy. Make the service selection more visual, branded, and sales-driven.

## Features

### Visual service selector

Cards/photos for each service:

- low fade;
- mid fade;
- high fade;
- taper;
- burst fade;
- beard trim;
- line up;
- kids cut;
- nail set;
- refill;
- design add-on;
- “por decidir.”

### Service packs

Encourage higher tickets through bundles:

- cut + beard;
- cut + eyebrows + beard;
- full clean;
- nail set + design;
- refill + repair;
- artist pack.

Multiple selected services can trigger discounts.

### Deposits

Use deposits to reduce no-shows.

### Booking

Simple flow:

1. choose service/service pack;
2. choose time or request time;
3. pay deposit if required;
4. receive confirmation;
5. partner receives order/booking by WhatsApp/email.

### Dashboard cards

Keep simple:

- sales this week;
- appointments coming;
- most requested service;
- repeat clients;
- no-show risk;
- best time/day;
- follow-up needed;
- recommended pack/promo.
