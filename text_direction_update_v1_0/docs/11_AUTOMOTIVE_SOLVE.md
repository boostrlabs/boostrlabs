# 11 — Automotive / SOLVE

## Status

SOLVE remains separate and live/important as a case study/reference. Do not migrate it into BOOSTR core automatically.

## How it relates to BOOSTR

SOLVE is an automotive vertical/system that proves BOOSTR can build industry-specific infrastructure.

BOOSTR automotive modules may include:

- lead capture;
- vehicle inventory;
- VIN decode;
- document checklist;
- deal status tracker;
- broker follow-up;
- AI concierge;
- WhatsApp qualification;
- partner-collected fee tracking;
- fixed commission per funded/closed vehicle;
- payment/direct collection records.

## AI Concierge rule

Automotive AI must not quote fixed prices as final. It may provide ranges/context and qualify the lead, then pass to human broker for negotiation.

AI handles:

- repetitive questions;
- after-hours qualification;
- budget;
- trade-in;
- urgency;
- payment/down range;
- location;
- docs needed.

Human broker closes the number.

## API notes

Useful future APIs:

- NHTSA VIN decoder;
- MarketCheck if budget allows;
- Google Business Profile Performance;
- Twilio/WhatsApp/Voice.

Avoid early credit/prequal APIs without legal/compliance review.
