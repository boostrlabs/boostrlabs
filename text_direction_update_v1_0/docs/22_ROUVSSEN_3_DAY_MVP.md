# 22 — Rouvssen / Janko WESTDETRO 3-Day MVP

## Goal

Enable Rouvssen to purchase music services from Janko/WESTDETRO through BOOSTR as soon as possible.

## Positioning

Do not present this as BOOSTR selling generic creative services.

Present it as:

- Janko Diorr / WESTDETRO Services powered by BOOSTR.
- Order, payment, instructions, and follow-up in one place.

## Minimum flow

1. Rouvssen opens a Smart Service Link.
2. Chooses a service.
3. Fills an intake form.
4. Pays via Stripe/payment link/direct method.
5. Receives confirmation.
6. Janko/BOOSTR receives order by email/WhatsApp.
7. Order is recorded.
8. Post-purchase page explains next steps.

## Services

- Beat Pack.
- Custom Production.
- Mix/Master Review.
- Creative Direction Session.
- Full Song Buildout.

## Data captured

- customer/artist name;
- WhatsApp;
- selected service;
- budget;
- urgency;
- references;
- links/files if any;
- payment method/status;
- delivery notes;
- order number.

## Keep simple

Do not wait for the full platform. A working Smart Service Link + order flow is enough to prove BOOSTR Network and Smart Checkout.
