# 07 — Verticals and Pilots

## Initial verticals

BOOSTR can theoretically serve any business with something to sell, but execution focuses on these initial verticals:

1. Artists / producers.
2. Brokers / car salespeople.
3. Content creators.
4. Barbers / nail artists / beauty.

Dashboard direction: mobile-first, app-like, red/black/y2k, very visual, low text, friendly, non-corporate.
