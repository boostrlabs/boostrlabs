# NNE Community Frontend

Starter frontend for the NNE Community MVP.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:5173`.

## Included

- React
- TypeScript
- Vite
- Responsive dark-luxury interface
- Dashboard
- Quests
- Listening trivia
- Rewards
- Feed
- Profile
- Mock data

Read `PROJECT_BRIEF_FOR_CODEX.txt` before implementing the backend.
