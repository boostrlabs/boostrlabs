# Hummus FL — Source Assets

Carpeta canónica para subir assets originales en alta calidad de Hummus Mediterranean Food.

## Regla principal

Subir aquí los archivos originales, sin comprimir y sin reemplazar nombres útiles.

## Nombres recomendados

- `hummus-logo-main.png`
- `hummus-logo-alt.png`
- `hummus-menu-current-01.png`
- `hummus-menu-current-02.png`
- `hummus-branding-old-01.png`
- `hummus-yelp-food-01.jpg`
- `hummus-google-food-01.jpg`
- `hummus-facebook-interior-01.jpg`
- `hummusasset1.jpg`
- `hummusasset2.jpg`
- `hummusasset3.jpg`

Los nombres `hummusassetX` también son válidos. Se analizarán visualmente y luego se clasificarán por uso.

## No subir aquí

- versiones comprimidas del navegador;
- outputs finales de Johanka;
- entregas de BOOSTR Missions;
- archivos duplicados sin diferencia real.

## Destino posterior

Después del análisis, los archivos se mapearán a:

- logo y branding;
- food assets;
- menú;
- hero/landing;
- Smart Links;
- QR;
- social media;
- ambiente/interior;
- evidencia y referencias.
